# Анализатор истории Bash (Bash History Analyzer)

Инструмент командной строки (CLI) на Python и SQLite для анализа вашей истории bash. Он помогает выявить часто используемые команды, найти возможные опечатки и предложить полезные псевдонимы (aliases).

## Возможности

- **Парсинг и хранение**: Читает файл `~/.bash_history`, разбирает команды и сохраняет их в локальную базу данных SQLite для анализа.
- **Топ команд**: Определяет 50 самых часто используемых команд.
- **Обнаружение опечаток**: Находит редкие команды, которые очень похожи (расстояние Левенштейна < 2) на частые команды (например, `gti` вместо `git`).
- **Предложения псевдонимов**: Предлагает короткие алиасы для длинных команд (> 15 символов), которые вы используете часто (> 10 раз).

## Требования

- Python 3.6+
- Стандартные библиотеки Python (`sqlite3`, `difflib`, `os`, `sys`, `re`)

## Установка

Клонируйте репозиторий:

```bash
git clone https://github.com/galimov-i/bash-history-analyzer.git
cd bash-history-analyzer
```

## Использование

Запустите анализатор с настройками по умолчанию (читает из `~/.bash_history`):

```bash
python3 main.py
```

Или укажите путь к файлу истории вручную:

```bash
python3 main.py /путь/к/вашему/.bash_history
```

## Как это работает

1.  **Парсинг**: Инструмент читает файл истории построчно. Игнорируются временные метки (строки, начинающиеся с `#`) и строки, начинающиеся с пробела (обычно такие команды не сохраняются в истории bash).
2.  **Хранение**: Разобранные команды сохраняются в `bash_history.db`.
3.  **Анализ**:
    -   **Частота**: Подсчитывает количество вхождений полных команд.
    -   **Опечатки**: Сравнивает редкие базовые команды с частыми, используя `difflib` для поиска различий в один символ.
    -   **Алиасы**: Фильтрует длинные и частые команды.

## Структура проекта

- `main.py`: Точка входа CLI и парсер файлов.
- `db_manager.py`: Отвечает за операции с базой данных SQLite.
- `analyzer.py`: Содержит логику анализа (частота, опечатки, алиасы).
- `sample_history.txt`: Пример файла истории для тестирования.
